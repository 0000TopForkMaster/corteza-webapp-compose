## Project setup

- commands :
```
yarn global add @vue/cli
yarn install
```
- create a copy of "config.latest.js" to "config.js".


### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
